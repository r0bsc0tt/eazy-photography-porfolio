Eazy Photography Portfolio creates a custom post type for eazy-photos and some custom taxonomies to organize your portfolio. 

== Description ==
Eazy Photography Portfolio creates a custom post type for eazy-photos. 
Two custom taxonomies are registered on activation, one for collections, the other for categories.

== Installation ==
1.  Install and activate Eazy Photography Portfolio
2.  Navigate to the Settings page in the admin under Photos.
3.  Turn the settings on you want to use. 
4. Add your photography portfolio just like a post.
